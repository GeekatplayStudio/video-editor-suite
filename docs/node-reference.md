# Node Reference

## Timeline And Guide Nodes

### `GAPDirector`

Visual prompt timeline editor for LTX workflows. Use it when you need per-segment prompts, guide strengths, and timeline-aligned custom audio.

### `GAPDirectorGuide`

Applies `guide_data` generated by `GAPDirector` to positive and negative conditioning plus a latent.

### `GAPSequencer`

Places guide images at chosen frame positions in a latent timeline. Useful for first-frame, last-frame, and multi-keyframe workflows.

### `GAPKeyframer`

Directly replaces frame regions in an existing latent sequence. Best when you already know exact keyframe positions.

### `GAPMultiImageLoader`

Loads multiple images with batch or per-image output behavior and image resizing support.

## Media Input And Utility Nodes

### `GAPLoadVideoUI`

Interactive video loader with preview, trim, crop, resize, and audio extraction. Outputs `IMAGE`, `AUDIO`, duration, and frame count.

### `GAPLoadAudioUI`

Interactive audio loader with trim controls. Outputs `AUDIO` plus duration.

### `GAPSpeechLengthCalculator`

Estimates spoken duration from quoted text to help size clips and timelines before rendering.

## Editing And Export Nodes

### `GAPClipEditor`

Single-clip editing node with trim, retime, loop count, reverse, ping-pong playback, frame blending, audio gain, and fade controls.

### `GAPTransitionComposer`

Combines two clips with one of several visual transitions: cut, crossfade, wipe-left, wipe-right, slide-left, or slide-right. Also supports transition hold frames and audio crossfade.

### `GAPMotionTextFX`

Adds styled text overlays, freeze-frame inserts, and speed-ramp behavior to an existing clip while keeping audio length aligned.

### `GAPVideoExporter`

Output node that encodes `IMAGE` plus optional `AUDIO` into `mp4`, `mov`, `webm`, `mkv`, or `gif` with codec, pixel format, preset, and bitrate controls.
